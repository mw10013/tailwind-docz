# Introduction

Sandbox for tailwind.

For tailwind intellisense in JS objects (assumes trailing ,)
"tailwindCSS.experimental.classRegex": [":\\s*?[\"'`]([^\"'`]*).\*?,"]
Related [issue](https://github.com/tailwindlabs/tailwindcss-intellisense/issues/129#issuecomment-785763832
) with more details.
